Fitness Shop

A web application for a fitness shop built with Vite, React, and Bootstrap for the frontend, and Express, MongoDB, and Mongoose for the backend. The app features user authentication using tokens and refresh tokens, with functionalities such as a carousel, React sliders, modals, and admin models for managing products and orders. It simulates an online store where users can browse fitness products, add them to their cart, and complete the checkout process.
